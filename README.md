# isosim-server-vuejs

## Introduction
A WIP vue based replacement for iso server (iso_server.html) plain JavaScript implementation in isosim
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
